# data/processed

Output directory for all pipeline-generated artifacts. Not all files carry equal operational status.

## Artifact classification

### runtime_contract

Live artifacts consumed by the operational runtime. A clean rebuild regenerates all of these.

| File | Producer |
|---|---|
| `ontology_aligned.ttl` | Hand-curated T-Box (source of truth) |
| `abox_input.json` | `abox_input_builder.py` |
| `abox_merged.ttl` | `abox_merger.py` |
| `abox_canonical.ttl` | `abox_canonicalizer.py` |
| `abox_enriched.ttl` | `abox_graph_enricher.py` |
| `abox_linked.ttl` | `abox_link_completer.py` |
| `multilingual_lexicon.json` | `multilingual_lexicon_builder.py` |
| `schema_condensed.txt` | `schema_condenser.py` |
| `canonical_entity_map.json` | `abox_canonicalizer.py` |
| `abox_minted_entity_registry.json` | `abox_graph_sanitizer.py` |
| `abox_semantic_audit.json` | `abox_semantic_validator.py` |
| `canonical_anchors.json` | Hand-curated anchor definitions |

### accepted_project_operational_artifact

Per-manual artifacts for the retained reference project. Preserved across clean rebuilds; refreshed when the corresponding manual is re-extracted.

Naming pattern: `<prefix>_merged.ttl`, `<prefix>_abox_input.json`, `<prefix>_merged_uri_collision_report.json`, etc., where `<prefix>` is one of `a218`, `quick_ref`, `installation_manual`, `man_8070_err`, `variables_cnc`.

### debug_and_diagnostics

Transient audit and diagnostic outputs. May be overwritten by any rebuild run.

- `*_abox_debug/` directories — per-chunk extraction failures
- `*_abox_graphs/` directories — per-chunk TTL files

### gitignored (regenerable, large)

The following files are excluded from version control due to size. They are regenerated by the pipeline on every rebuild:

- `multilingual_lexicon.json` (41 MB)
- `canonicalization_report.json` (15 MB)
- `canonicalization_resolution_candidates.json` (9.1 MB)
- `abox_generation_manifest.json` and `*_abox_generation_manifest.json`
- `abox_minted_entity_registry.json`
- `canonical_entity_map.json`
- `*_density_report.json`
- `*_language_detection_report.json`
